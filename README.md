# Plan Les
#### Video Demo:  https://youtu.be/183mNuVcWB8
#### Description:
Plan Les is a comprehensive lesson planner web application designed for educators, tutors, and course coordinators. Ideally, teaching requires structure, and Plan Les provides a digital dashboard to organize courses, manage classes, and plan lessons effectively.

This project was built using Python (Flask), HTML, CSS, and JavaScript. It utilizes a SQLite database to ensure data persistence, allowing teachers to maintain their lesson plans across sessions.

## Features
- **User Authentication**: Secure registration and login for teachers.
- **Dashboard**: A central hub displaying all created courses.
- **Course Management**: Create and manage different courses or classes.
- **Lesson Planning**: Add detailed lesson plans for each course, including titles, topics, dates, and private notes.
- **Data Persistence**: All data is stored securely in a SQL database.

## Usage
1.  Install dependencies: `pip install -r requirements.txt`
2.  Run the application: `flask run`
3.  Register for an account and start planning!
